# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end **data analytics workflow** starting from raw data processing to final business insights. The project involves **data cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and reporting**.

Python was used for **data preprocessing and analysis**, PostgreSQL was used to **store and query the data**, and Power BI was used to **build an interactive dashboard**. The final insights were summarized in a **report and presentation created using Gamma**.

The goal of this project is to transform raw data into **meaningful insights that support data-driven decision making**.

---

## 📂 Dataset

The dataset contains structured data used for analysis and visualization.

**Dataset includes information such as:**

* Categories and sub-categories
* Numerical performance metrics
* Time-based variables
* Additional attributes relevant for analysis

The dataset was first analyzed in **Python** and then stored in a **PostgreSQL database** to run SQL queries for deeper insights.
